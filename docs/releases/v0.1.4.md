# 🚀 Release v0.1.4

## ✨ New Features
1. Replace lumberjack with custom rotate writer (rotator)
	- Các options như lumberjack vẫn được hỗ trợ nhưng thêm phần cho phép config format file log
	- Filename:   filename,
	- MaxSize:    maxSize,
	- MaxBackups: maxBackups,
	- MaxAge:     maxAge,
	- Compress:   compress
        ...
	- (mặc định: {basename}_{index}.log, những loại khác: {basename}_{time_format}_{index}.log: app_1.log, app_2.log, or app_20251021_1.log, app_20251021_2.log)

2. Tổ chức lại code thành các 3 packages nhỏ hơn, theo flow mới của logger
    logger/
    formatter/
    rotator/
